Projetos de cursos e mockados

  • RickAndMorty App - Projeto Android feito como um teste, parte de um processo seletivo que fiz para ingressar em uma empresa. Consiste em uma lista infinita com os personagens da série Rick & Morty.

    Tecnologias utilizadas:

    • Kotlin
    • Android SDK
    • Picasso
    • Retrofit
    • Paging Library
  • Pokeapp iOS - Projeto iOS para demonstrar o uso de criação de layout e chamadas http na plataforma móvel da Apple.

    Tecnologias utilizadas:

    • Swift
    • Swift UI
  • Rológio pomodoro - Projeto flutter para demonstrar o uso de programação reativa com flutter usando mobx

    Tecnologias utilizadas:

    • Flutter
    • Dart
    • Mobx
    • Flutter Provider
  • SpringRestApi-OSWork - Projeto SpringBoot feito durante um curso introdutório ao SpringBoot. Um simples CRUD usando a entidade "cliente"

    Tecnologias utilizadas:

    • Java
    • SpringBoot
    • Java Persistence API (JPA)
    • MySQL
